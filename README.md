# Vue.js 移动端项目——黑马头条

## 功能特性

- 首页
  + 频道列表
  + 文章列表
  + 频道编辑
- 文章详情
  + 基本信息展示
  + 文章评论
- 搜索
  + 联想建议
  + 搜索结果
- 问答
- 视频
- 我的
  + 我的收藏
  + 我的历史
  + 我的作品
- 登录
- 个人信息
- 个人页面
- 关注/粉丝
- 小智同学

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
